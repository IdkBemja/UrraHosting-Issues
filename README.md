# UrraHosting — Issues

Este repositorio está destinado **exclusivamente** para reportar bugs y problemas de [UrraHosting](https://urrahosting.cl).

> **No se aceptan solicitudes de nuevas funcionalidades, preguntas de soporte general ni discusiones ajenas a errores o problemas con la plataforma.**

## ¿Cómo reportar un issue?

1. Ve a la pestaña [**Issues**](../../issues) y haz clic en **New issue**.
2. Selecciona la categoría que corresponde:
   - 🐛 **Bug Report** — algo que no funciona como debería (error inesperado, comportamiento incorrecto).
   - ⚠️ **Problem Report** — un problema detectado en la plataforma (degradación de rendimiento, proceso fallido, etc.).
3. Completa **todos** los campos del formulario, incluyendo el nivel de severidad, la descripción detallada y las evidencias.

## Niveles de severidad

| Nivel | Descripción |
|-------|-------------|
| 🔴 **Crítico** | El servicio está caído o es completamente inutilizable. |
| 🟠 **Alto** | Una funcionalidad principal está afectada gravemente. |
| 🟡 **Medio** | Una funcionalidad está afectada pero hay forma de continuar. |
| 🟢 **Bajo** | Problema menor o cosmético que no bloquea el uso. |
